PROJETO INTEGRADOR

TRANCA LIMITADORA DE ACESSO COM RFID

Introdução

Nosso objetivo é trazer a acessibilidade para a segurança no trabalho usando do Arduino Uno e RFID para elaborar uma tranca com display de acesso para ser utilizada em áreas de risco como salas de produtos químicos ou locais de exposição à vibração, pressão, ruído, temperaturas extremas, radiações não ionizantes e radiações ionizantes – como salas de exames de radiologia. Considerando essas problemáticas, foi elaborado um sistema de segurança utilizando RFID e o microcontrolador Arduino UNO para limitar o acesso a áreas de risco à integridade física de seus usuários além de um sistema de alarme de segurança caso haja mais tentativas de não autenticação.

Materiais

Utilizamos como materiais os seguintes itens: Placa Arduino Uno Atmega328 Smd, cabo USB, buzzer passivo 5v, protoboard com jumper Mxm/Mxf e leitor Rfid Rc522 com cartão e tag 13.56mhz. O software utilizado – Arduino IDE – para sua programação possui código aberto, sendo assim, há a possibilidade de realizar o download do diagrama de circuito, comprar todos os componentes e criar seu próprio Arduino. O software também pode ser lido em Windows, MacOS e Linux e faz-se possível utilizar de cabo USB, sem necessidade de porta serial.
